## 核心投资逻辑

### 两个底层信念

1. **大公司的市值长期一定往上走**——通胀 + 经济发展 + 龙头地位 = 大公司不会死
2. **大公司短期下跌是事件驱动的**——行业低谷、黑天鹅事件导致暴跌，但龙头一定会回来

**核心推论**：大公司 + 低价买入 = "我只会亏时间，不会亏钱"

### 不看好坏，只看高低

用大市值筛选解决了"公司好不好"的问题，剩下只需判断"价格高还是低"。

## 买入策略：支撑位驱动的分批计划（核心方法论）

### 选股
看市值排行，找近几年跌幅较大的大公司，市盈率处于历史低位。

### 买点
看历史K线找关键支撑位（前期低点、平台位置），到支撑位附近观察盘面——撑住了买，撑不住等下一个。

### 分批规则（铁律中的铁律）

| 规则 | 说明 |
|------|------|
| 资金分10份 | 从当前价附近第一个支撑位开始 |
| 每个价位有K线依据 | 前期低点/成交密集区/52周低点/整数关口 |
| 撑住买1份 | 撑不住不买，等下一个更低支撑位 |
| 绝不加倍 | 到最低点也严格每次1份 |
| 间距不均匀 | 由K线形态决定，禁止等差数列 |
| 10次用完不买 | "能吃到多少就吃多少" |
| 买5只分散 | "一个股票不保险" |

**支撑位类型（优先级）**：
1. 前期明确低点（多次触及未破更强）
2. 成交密集区/平台（横盘整理长的区间）
3. 52周低点（最后一道防线）
4. 整数关口（心理价位，辅助参考）

## 卖出策略

| 卖出信号 | 触发条件 |
|---------|---------|
| 牛市结束 | 第三波冲高放量但冲不动 → 全部清仓 |
| 市盈率过高 | 涨到历史高位区间 → 卖出 |
| 快速冲顶 | 短期急速拉升有冲顶感 → 可做T减仓 |

**怎样判断"第三波冲不动"**：
1. 数波次：牛市通常2-3波主升浪，到第三波警惕
2. 看量价背离：指数创新高但成交量没跟上
3. 看连续大阳后缩量/长上影 → 力竭信号
