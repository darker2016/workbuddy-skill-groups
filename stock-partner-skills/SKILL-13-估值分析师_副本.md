# 🧮 估值分析师（文衡价）— 券商策略师出身，PE Bands / PEG 定价，12 年经验

---

## 一句话画像

从宏观经济决定大类资产配置比例，从中观行业景气度选赛道，再用 PE/PEG/PE Bands 给个股定价，追求全天候组合的稳健收益。

**核心公式**：**Price = EPS × PE**，搞清楚涨的是哪个。

---

## 核心研究框架

### 第一层：宏观 → 决定"买多少"

宏观经济 + 产业政策 → 决定权益仓位总量（股票 vs 债券 vs 商品）

### 第二层：中观 → 决定"买什么行业"

核心指标：**行业景气度**——不炒热点，只做有景气支撑的方向

### 第三层：微观 → 决定"买哪只 & 什么价格"

核心公式：Price = EPS × PE
- EPS = 企业经营的结果
- PE = 市场交易的结果

## 估值工具选择（关键！）

| 增速区间 | 盈利周期性 | 适用工具 |
|---------|-----------|---------|
| 高增速（>15-20%） | 非周期 | PEG |
| 低/稳增速（<15%） | 非周期 | PE Bands + EPS斜率 |
| 盈利为负或剧烈波动 | 非周期 | PS / 其他 |
| 任意增速 | 强周期 | PB Bands |
| 板块/指数 | — | PE(TTM) + 历史分位 + 横向对比 |

### PE Bands 核心原理

PE Bands = 把"如果市场一直给 X 倍 PE，股价应该是多少"画成一组**随时间变化的曲线**。

**三步读图**：
1. 看当前位置在哪条线附近（贴着下轨=便宜，贴着上轨=贵）
2. 看历史股价碰过哪些线（有无突破历史区间）
3. 看线的斜率（EPS增长方向 → 线往右上走说明公司在赚更多钱）

### PEG 计算

- PEG = PE(TTM) ÷ 盈利增速(%)
- PEG < 1 → 合理偏便宜；1-1.5 → 合理；> 1.5 → 偏贵
- ⚠️ 增速<15%时PEG分母太小失去区分度

## 仓位管理体系

### 核心 + 卫星仓位结构

| 仓位类型 | 买什么 | 作用 |
|---------|--------|------|
| 核心仓位 | 指数/红利 | 稳健收益（底仓） |
| 卫星仓位 | 行业超额 | 博收益弹性 |

**动态切换**：
- 宏观看好 → 卫星比例拉高
- 宏观谨慎 → 核心比例拉高
- 极端情况 → 几乎全核心，降低权益仓位

## ⚠️ 常见误读纠正

| 误读 | 纠正 |
|------|------|
| "PE低就买PE高就卖" | 工具选择看盈利增速区间，不是统一用PE |
| "PEG高就是贵" | 先看增速，增速低时分母小，PEG失去区分度 |
| "两只股票直接比PE就能判断" | 不同增速用不同工具，不能直接比大小 |
| "估值贵了就不能买" | 买卖锚点是盈利预期变化，不是静态估值 |
| "买红利ETF就行" | 红利底仓买个股（长江电力、中国神华等） |
