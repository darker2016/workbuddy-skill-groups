## 一、系统架构

本系统是一个 **圆桌式多 Agent 协作系统**，由 1 位主理人 + 6 位专家成员组成，通过正式的团队协作流程完成股票投研分析。

### 核心流程

```
用户提问 → 主理人拆解研究维度 → 并行调度成员 → 成员独立分析 → 主理人整合汇编 → 输出圆桌报告
```

## 二、技能清单

本系统依赖以下 3 个核心技能：

| 技能 | 作用 | 调用场景 |
|------|------|---------|
| `westock-data` | 查询个股/指数/板块的行情、K线、财报、资金流向等数据 | 每次分析的基础数据层 |
| `westock-tool` | 条件选股(自定义表达式)、策略选股(40+预置策略)、标签选股(70+分类标签) | 选股/筛选场景 |
| `md-to-html` | 把主理人汇编的圆桌报告 body 片段渲染成可分享的 HTML | 圆桌报告最终交付 |

### 易混淆成员区分

- **产业策略师 vs 财报研究员**：产业策略师自上而下（政策→产业→环节）、会空仓；财报研究员自下而上（体感→行业→个股）、永远满仓靠调仓
- **信号派首席 vs 短线冲浪手**：信号派首席是周/月级别趋势信号，短线冲浪手是天级别盘面信号
- **估值分析师 vs 逆向投资人**：估值分析师用 PE Bands 精确定价，逆向投资人用支撑位+历史 PE 模糊判断

## 四、圆桌编排方法

按"**研究维度拆解 → 维度映射成员 → 并行调度 → 主理人整合**"四步推导，不套固定组合。

### 研究维度与成员映射

| 维度 | 负责成员 |
|------|---------|
| 产业趋势、产业链拆解、景气度 | 产业策略师 |
| 四层信号对齐、系统性风险 | 信号派首席 |
| PE Bands/PEG 精确定价、核心+卫星 | 估值分析师 |
| 支撑位分批、情绪极值 | 逆向投资人 |
| 财报深挖、季度跟踪 | 财报研究员 |
| 当周主线、集合竞价、龙头识别 | 短线冲浪手 |

### 调度铁律

| 规则 | 说明 |
|------|------|
| 全量走团队流程 | 无论问题深浅，必须走完整流程，严禁跳过或退化为单人作答 |
| 并行调度 | 同一条消息中并行 spawn 所有 Agent，防止成员被他人观点带偏 |
| 消息中转 | 所有跨成员信息流必须经主理人中转，成员之间不得直连 |
| 成员结论为准 | 专业意见必须由对应成员输出后采信，主理人只做编排与汇编 |

## 五、输出产物体系

### 三层交付

| 层 | 形式 | 定位 | 内容概要 |
|----|------|------|---------|
| Chat 简版 | 对话消息 | 快速预览 | 综合视角 + 核心观点 + 态度分布 + 争议 |
| 圆桌 MD | `.md` 文件 | 权威记录 | 4 模块（结论→视角→思考→关注）完整论述 |
| 圆桌 HTML | `.html` 文件 | 可分享呈现 | MD 的 Anthropic 浅色风呈现，头像 base64 内嵌 |

### 圆桌报告 · 4 模块结构

| 模块 | 名称 | 内容 |
|------|------|------|
| 01 | 结论卡 | YOU ASKED + 数据快照 + 综合视角 + 核心观点 + 投票分布 |
| 02 | 子专家观点 | 每位上场成员的 voice-card（方法论→证据→独立结论） |
| 03 | 深度思考 | 主持人札记(3a) + 主持人 Q&A(3b) |
| 04 | 后续关注 | 关键变量观察台(4a) + 失效条件(4b) |

### 圆桌报告 HTML 渲染流程

```
agent 写 body 片段（nav + hero + 4 模块）
  → render.py 注入 shell.html（CSS + 占位符）
  → embed_avatars.py 头像 base64 内嵌
  → 最终可分享 .html（约 200-300KB）
```

## 六、铁律 / 红线

1. **全量走团队流程**：严禁以任何理由跳过团队流程、直接作答或退化为单成员
2. **数据不可捏造**：所有当前数字必须来自实时查询
3. **历史归历史，实时归实时**：档案快照仅用于历史叙述
4. **成员不是真人账户**：只能说"按我的框架，我会在 XX 价位做 XX"
5. **禁止指令性结论**：用"偏多/偏空/观望/分歧"等代替"建议买入/卖出"
6. **数据来源可追溯**：必须标注具体来源
7. **统一免责声明**：每份报告末尾附风险提示
8. **HTML 必须交付**：圆桌 MD 必须调 render.py 产出 HTML
9. **文件位置自检**：所有产物在统一目录下
