#### 典型真题解析

**例题1（编程题 — 基础计算）**：编写程序，输入一个正整数 n，输出 1! + 2! + ... + n! 的值。

```python
def factorial(x):
    result = 1
    for i in range(1, x + 1):
        result *= i
    return result

n = int(input("请输入正整数 n："))
total = 0
for i in range(1, n + 1):
    total += factorial(i)
print(total)
```

**例题2（编程题 — 字符串处理）**：输入一行字符串，统计其中英文字母、数字、空格和其他字符的个数。

```python
s = input("请输入字符串：")
letters = digits = spaces = others = 0
for ch in s:
    if ch.isalpha():
        letters += 1
    elif ch.isdigit():
        digits += 1
    elif ch.isspace():
        spaces += 1
    else:
        others += 1
print(f"字母{letters}个，数字{digits}个，空格{spaces}个，其他{others}个")
```

**例题3（编程题 — 文件操作）**：读取文件 data.txt，将每一行的数字（每行一个整数）累加求和，并输出到 result.txt。

```python
with open('data.txt', 'r') as f:
    lines = f.readlines()
total = 0
for line in lines:
    line = line.strip()
    if line:
        total += int(line)
with open('result.txt', 'w') as f:
    f.write(str(total))
```

**VLOOKUP 函数详解（必考）**：
```
=VLOOKUP(查找值, 表格数组, 列序号, [匹配方式])
= VLOOKUP(A2, $E$2:$F$100, 2, FALSE)
```
- 第四个参数 FALSE 表示精确匹配（必考）
- 查找值必须在表格数组的第一列
- 表格数组用绝对引用 `$E$2:$F$100` 锁定
