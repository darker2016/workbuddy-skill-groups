# 采集报告输出模板

> 被灵阅技能引用，定义每条采集内容的输出格式。

## 单条内容模板

```markdown
## 📰 [文章/视频标题]

**来源平台**：微信公众号 / X / YouTube / B站 / GitHub
**来源URL**：[可点击跳转的完整链接]
**发布时间**：YYYY-MM-DD
**作者/账号**：[名称]
**热度指标**：阅读/点赞/评论数（如有）

### 核心摘要
（50-150字，用自己的话概括，禁止复制粘贴）

### 关键要点
- ✅ 要点1（20字以内）
- ✅ 要点2
- ✅ 要点3

### 金句摘录
> 原文中最有价值的一句话
> —— 来源：[来源URL 或 来源账号]

### 适合做视频的点
（说明做成视频的最佳切入角度）

**质量评分**：⭐⭐⭐⭐（推荐做视频 / 一般 / 不推荐）
**评分理由**：一句话说明为什么这个分
```

## 采集报告整体结构

```markdown
# 采集报告：[主题关键词]

**采集时间**：YYYY-MM-DD
**采集范围**：近7天 / 近一月
**平台覆盖**：微信公众号、X、YouTube、B站、GitHub
**有效内容数**：N 条

---

[内容1]

---

[内容2]

---

## 采集小结

**推荐做视频的内容**：
1. [标题] — 理由
2. [标题] — 理由

**不推荐的内容**：
- [标题] — 原因（重复/过时/价值低）
```
