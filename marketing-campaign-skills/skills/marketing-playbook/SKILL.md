# 营销战役团队手册 (Marketing Playbook)

本手册覆盖营销战役团队的全部 **8 项核心能力**。主理人根据用户需求自动选择合适的工作流。

---

## 1. 内容创作 (Content Creation)

创建各种类型的营销内容。

**触发词**: 写博客、营销文案、白皮书、案例研究

### 内容类型

| 类型 | 长度 | 目的 |
|------|------|------|
| 博客文章 | 800-2000 字 | SEO、思想领导力 |
| 社交帖子 | 50-280 字 | 品牌曝光、互动 |
| 白皮书 | 3000-5000 字 | 引流、建立权威 |
| 案例研究 | 1000-2000 字 | 社会证明、销售赋能 |
| 邮件通讯 | 200-500 字 | 培育、留存 |
| 着陆页文案 | 500-1500 字 | 转化 |

### 创作框架

**AIDA**：Attention(标题) → Interest(共鸣) → Desire(价值+证据) → Action(CTA)

**PAS**：Problem(痛点) → Agitate(放大) → Solution(方案)

### 四原则

1. 先说对读者的价值，不是你多厉害
2. 一篇内容一个核心信息
3. 用具体数据代替形容词
4. CTA 要明确

## 2. 快速起草 (Draft Content)

快速产出营销内容草稿。

**触发词**: 帮我写、起草、draft

### 工作流程

1. **确认**: 类型 + 受众 + 核心信息 + 语气
2. **起草**: 标题选项(2-3个) + 正文 + CTA + SEO关键词 + 配图建议
3. **迭代**: 根据反馈调整

## 4. 品牌审核 (Brand Review)

审核营销内容是否符合品牌规范。

**触发词**: 品牌审核、品牌一致性、内容审查

### 审核维度

| 维度 | 关注点 |
|------|--------|
| 语气调性 | 是否匹配品牌属性？有无偏移？ |
| 术语用语 | 品牌首选术语？禁止词语？ |
| 信息一致性 | 与品牌支柱和定位是否一致？ |
| 合规检查 | 未证实声明？竞品对比合规？免责声明？ |

### 输出模板

```markdown
## 品牌审核: [内容标题]
### 总评: 🟢 优秀 / 🟡 需调整 / 🔴 需重写

### 问题清单
| # | 位置 | 类别 | 严重程度 | 问题 | 建议修改 |

### 做得好的: [亮点]
```

## 5. 营销竞品分析 (Competitive Brief)

分析竞品的营销策略和市场定位。

**触发词**: 竞品营销、竞品策略、营销对标

### 分析维度

- **定位和信息**: 品牌定位 + 价值主张 + 目标客户
- **内容策略**: 类型/频率 + 话题 + SEO 关键词
- **渠道策略**: 平台 + 发布频率 + 互动数据 + 付费广告
- **产品营销**: 定价页 + 功能展示 + 案例研究

### 输出模板

```markdown
## 营销竞品分析: [竞品]
### 快照对比表
| 维度 | 他们 | 我们 | 差距 |

### 他们做得好的 → 我们的机会 → 建议行动
```

## 7. SEO 审计 (SEO Audit)

对网站进行全面的 SEO 审计。

**触发词**: SEO 审计、SEO 分析、关键词机会

### 审计维度

**技术 SEO**: 加载速度 / 移动适配 / HTTPS / Sitemap / 结构化数据 / 404

**页面 SEO**: 各页面的问题、严重程度和修复建议

**关键词机会**:
| 关键词 | 难度 | 搜索量 | 当前排名 | 推荐内容类型 |

**内容缺口**:
| 主题 | 重要性 | 推荐格式 | 优先级 | 工作量 |

### 行动计划

- **Quick Wins (本周)**: 2 小时内可完成、立即见效
- **Strategic (本季度)**: 投入大但长期价值高

## 8. 邮件序列 (Email Sequence)

设计和撰写邮件营销序列。

**触发词**: 邮件序列、EDM、邮件营销、nurture

### 序列类型

| 类型 | 目的 | 典型长度 |
|------|------|---------|
| 欢迎序列 | 新用户 onboarding | 5-7 封 |
| 培育序列 | 线索孵化 | 6-10 封 |
| 促销序列 | 限时活动 | 3-5 封 |
| 重激活 | 沉默用户唤醒 | 3-4 封 |
| 教育序列 | 功能引导 | 4-6 封 |

### 输出模板

```markdown
## 邮件序列: [名称]

### 概览: 目标 / 受众 / 触发条件 / 频率

### 邮件 [N]: [标题]
**发送时间**: 触发后 [X]
**主题行**: [A] | [B]
**正文**: [完整文案]
**CTA**: [按钮文案]

### 关键指标
| 指标 | 行业基准 | 目标 |
|------|---------|------|
| 打开率 | [X]% | [X]% |
| 点击率 | [X]% | [X]% |
| 退订率 | < [X]% | < [X]% |
```
