## 1. 招聘漏斗 (Recruiting Pipeline)

**触发词**: 招聘进度、候选人状态、招聘效率

### 漏斗阶段
| 阶段 | 说明 | 关键动作 |
|------|------|---------|
| 寻源 | 识别并触达 | 个性化触达 |
| 筛选 | 电话/视频初筛 | 评估基本匹配 |
| 面试 | 结构化面试 | 多轮评估 |
| 讨论 | 团队决策 | 校准反馈 |
| Offer | 发出录用通知 | 薪酬包、谈判 |
| 入职 | Offer 已接受 | 转入入职流程 |

### 关键指标
- Pipeline 速度（每阶段天数）/ 转化率 / 渠道效果 / Offer 接受率 / 招聘周期

### 输出模板

```markdown
## 招聘漏斗 — [岗位/时间范围]

### 漏斗数据
| 阶段 | 候选人数 | 转化率 | 平均天数 |

### 瓶颈分析 → 优化建议
```

## 2. 面试方案设计 (Interview Prep)

**触发词**: 设计面试、面试问题、评分标准

### 五原则
1. **结构化** — 同岗所有候选人使用相同问题
2. **能力导向** — 问题映射到具体能力项
3. **行为证据** — STAR 格式
4. **多元面试官** — 降低偏见
5. **评分卡** — 1-4 分量化评估

### 输出模板

```markdown
## 面试方案: [岗位]

### 能力模型
| 能力 | 权重 | 评估轮次 |

### 面试安排
| 轮次 | 面试官 | 时长 | 重点 |

### 问题库（每项能力 2-3 个 STAR 问题 + 追问 + 优秀回答特征）

### 评分卡
| 4 超出预期 | 3 达到预期 | 2 部分达到 | 1 未达预期 |
```

## 3. Offer 起草 (Draft Offer)

**触发词**: 起草 Offer、录用通知、发 Offer

### 输出模板

```markdown
## 录用通知
**候选人:** [姓名] | **岗位:** [职位] | **入职日期:** [Date]

### 薪酬包
| 组成 | 金额 | 说明 |
|------|------|------|
| 基本工资 | ¥[X]/月 | 税前 |
| 年终奖 | [X] 个月 | 根据绩效 |
| 签字费 | ¥[X] | 一次性 |
| 股权 | [X] 股 | [X] 年归属 |

### 福利 + 条款（试用期/工作地点/工时）
### 回复截止: [日期]
```

**注意**: 试用期工资不低于正式的 80%（劳动法合规）；Offer ≠ 劳动合同。

## 4. 入职引导 (Onboarding)

**触发词**: 入职计划、新人引导、30/60/90

### 输出模板

```markdown
## 入职计划: [姓名] — [岗位]
**入职日期:** [Date] | **经理:** [人] | **Buddy:** [人]

### 入职前（开通账号、采购设备、欢迎邮件、指定 Buddy）

### Day 1
| 时间 | 活动 | 对接人 |
|------|------|--------|

### Week 1（合规培训/文档阅读/1:1/旁听会议/第一个小任务）

### 30天: 学习融入 → 60天: 独立贡献 → 90天: 全面产出
```

**原则**: Day 1 不超载 / 指定 Buddy / 目标具体可衡量。

## 5. 薪酬分析 (Comp Analysis)

**触发词**: 薪酬对标、这个岗位给多少、Offer 有竞争力吗

### 总薪酬框架
- 基本工资 + 股权(RSU/期权) + 奖金(年终/签字费) + 福利(五险一金等)
- 关键变量: 岗位 / 级别 / 地点(北上广深 vs 二三线) / 公司阶段 / 行业

### 输出模板

```markdown
## 薪酬分析: [岗位/范围]

### 市场基准
| 分位 | 基本工资 | 股权 | 总薪酬 |
|------|---------|------|--------|
| P25 | ¥[X] | ¥[X] | ¥[X] |
| P50 | ¥[X] | ¥[X] | ¥[X] |
| P75 | ¥[X] | ¥[X] | ¥[X] |

### 薪酬带分析（如提供数据）
### 建议（薪酬/股权/留存风险）
```

**原则**: 看总薪酬非仅基薪 / 标注地域 / 默认 ¥ / 数据保密。

## 6. 组织规划 (Org Planning)

**触发词**: 组织架构、团队扩张、Headcount 规划

### 健康基准

| 指标 | 健康范围 | 警告信号 |
|------|---------|---------|
| 管理幅度 | 5-8 人 | <3 或 >12 |
| 管理层级 | 4-6 层 | 过多=决策慢 |
| IC:经理比 | 6:1 到 10:1 | <4:1=头重脚轻 |
| 团队规模 | 5-9 人 | <4 孤立, >12 难管 |

### 输出模板

```markdown
## 组织规划: [范围]

### 当前状态（总人数/层级/幅度/IC:经理比）
### 组织图（文本缩进版）
### 健康度评估表
### Headcount 规划（岗位/数量/优先级/成本/入职时间）
### 分阶段招聘路线图
```

## 7. 绩效评估 (Performance Review)

**触发词**: 绩效评估、自评模板、经理评估、校准

### 自评模板
- 关键成果（STAR 格式）+ 目标回顾（达成/超越/未达）+ 成长/挑战/下周期目标

### 经理评估模板
- 整体评级(1-5) + 优势(附事例) + 发展领域(附指导) + 发展计划(技能/当前/目标/行动)

### 校准文档

```markdown
## 绩效校准 — [团队] — [周期]

### 评级分布
| 评级 | 人数 | 占比 | 公司目标 |
|------|------|------|---------|
| 5 卓越 | [X] | [X]% | ~5% |
| 4 优秀 | [X] | [X]% | ~20% |
| 3 达标 | [X] | [X]% | ~60% |
| 2 待改进 | [X] | [X]% | ~10% |
| 1 不达标 | [X] | [X]% | ~5% |

### 晋升候选人
| 姓名 | 当前级别 | 提议级别 | 理由 |
```

## 8. 人员分析 (People Report)

**触发词**: 人员报告、流失分析、留存率、eNPS

### 指标体系
- **留存**: 总流失率/主动流失率/平均任期/离职风险
- **多样性**: 各级别代表性/晋升率差异/薪酬公平性
- **参与度**: 调查分数/eNPS/参与率
- **效率**: 人均营收/管理幅度效率/招聘周期

### 输出模板

```markdown
## 人员分析报告 — [时间范围]
### 核心指标（本期/上期/趋势/行业基准）
### 关键洞察（附数据）
### 风险信号
### 建议行动
```

## 9. 政策查询 (Policy Lookup)

**触发词**: 年假怎么算、报销政策、远程办公规定

### 常见主题
- 休假（年假/病假/产假/婚假/丧假）
- 福利（五险一金/商业保险/体检/餐补）
- 薪酬（发薪周期/年终奖/报销）
- 远程（居家办公/异地办公/设备补贴）
- 差旅（预订标准/报销/审批）
- 行为（行为准则/反骚扰/利益冲突）
- 成长（培训预算/会议/学费报销）

### 输出模板

```markdown
## 政策: [主题]
### 简答: [1-2句直接回答]
### 详情: [通俗语言解释]
### 例外/特殊情况
### 联系人: [超出范围的问题找谁]
### 来源: [文档名称/章节]
```

**原则**: 找不到就明确说找不到 / 用通俗语言 / 法律问题建议咨询专业人士。
