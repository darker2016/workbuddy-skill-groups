## 预设工作流

- **Workflow 1**：全链路数据分析（数据库→分析→可视化→报告）
- **Workflow 2**：CSV/Excel数据画像
- **Workflow 3**：知识库增强分析（文档+数据综合分析）
- **Workflow 4**：纯知识问答
