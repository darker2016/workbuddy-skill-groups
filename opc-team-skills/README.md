# OPC一人公司专家团

> WorkBuddy Expert Team 上架包 · 基于《一人企业方法论》第三版

## 团队简介

OPC一人公司专家团是基于 [《一人公司方法论》](https://github.com/easychen/opc-methodology) 的 9 阶段共创团队，由 1 位主理人 + 8 位领域成员组成，陪伴用户完整走完一人公司从「资源盘点 → 经营复盘」的全流程。

## 适用人群

- 想从零开始规划一个一人公司 / 独立项目的人
- 已经在做但卡在某个阶段（找不到利基、不知道怎么把价值说清楚、不会设计 MVP…）
- 已有产品在跑，想做一次系统的经营复盘

## 与用户的协作流程

**建盘期（01–07，线性、一次走完）**

```
01 资源盘点 → 02 利基定位 → 03 价值主张 → 04 商业模式
                                            ↓
                             06 MVP 设计 → 07 转化闭环
```

阶段 07 完成后**进入执行阶段**——按转化路径实际去做，专家团在用户卡住或想沉淀时再回归。

**运营循环（08 / 09，可多次触发）**

- 想系统化重复出现的成果 → 调度 **资产沉淀师 沉墨**
- 卡住、做周期性回顾 → 调度 **经营复盘师 复明**

## 技术参数

- 目录契约：所有专家共享 `opc-doc/`（state / inputs / outputs 三个子目录）
- 主理人遵循「团队协作铁律」——所有成员调度走 TeamCreate → 调度 → 回传流程，禁止主理人代写成员产出
- 每个成员预加载对应 `skills/opc-*` 详细方法论
- 默认中文，遵循「教学 / 引导 / 直通」三种交互模式
