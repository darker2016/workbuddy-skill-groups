## 模考组织方案

### 模考频次安排

| 冲刺时长 | 模考安排 |
|---------|---------|
| 4周冲刺 | 第2周1次摸底 + 第3周1次强化 + 第4周1次最后冲刺 |
| 6周冲刺 | 第2周1次摸底 + 第3周1次 + 第4周1次 + 第5周2次 + 第6周1次 |
| 8周冲刺 | 每周1次，最后两周每周2次 |

### 模考执行流程

**考前准备：**
1. 准备完整真题卷（阅读+写作+听力，不含口语）
2. 准备答题卡（从官网下载打印）
3. 准备计时器
4. 清理桌面，模拟真实考场环境

**模考执行：**
```
阅读与写作（60分钟）→ 休息5-10分钟 → 听力（约30分钟）
```
- 严格按照官方时间执行，中途不暂停
- 不建议提前看题目或暂停重听
- 家长/老师只负责计时，不提供任何帮助

**口语模考（单独安排）：**
- 找一位同伴进行双人口语模拟
- 家长/老师扮演考官
- 按照官方流程走完 Part 1 → Part 2 个人 → Part 2 互动
- 录下全过程，用于复盘

## 答题卡填涂训练

### 常见填涂错误

1. 答案写对但填涂位置不对（错位）
2. 用圆珠笔而不是2B/HB铅笔
3. 修改时擦不干净，机器识别两个答案
4. 姓名/考号填写错误
5. 听力 Part 2 填空大小写不规范

### 训练方法

- 第1次模考：只用答题卡做题
- 第2次模考：严格模拟考试流程（发卷 → 填涂 → 交卷）
- 每次模考后检查答题卡填涂质量
- 纠正填涂速度（不能超过做题时间的10%）

## 时间管理训练

### 三步法训练

| 步骤 | 方法 | 目的 |
|------|------|------|
| ① 摸底 | 不限时做一套完整卷，记录每部分实际用时 | 找到基准线 |
| ② 对标 | 与建议时间对比，找出"时间黑洞" | 识别瓶颈 |
| ③ 限时训练 | 针对"时间黑洞"进行限时专项训练 | 提速突破 |

### 专项限时训练示例

| 问题 | 训练方法 |
|------|---------|
| Reading Part 4 花太久 | 每天2篇句子还原，每篇限时7分钟 |
| 作文写不完 | 每天5分钟写一篇邮件，逐次缩短时间 |
| 检查时间不够 | 每次做题时在每题上写"预计用时"，做完对比 |

### 考场节奏训练

第1-2周模考时，每做完一个 Part 报时；第3-4周模考时，让学习者自己掌握节奏，不再报时，模拟真实考场。

## 考前心理建设

### 考前倒计时

| 时间节点 | 行动要点 |
|---------|---------|
| **T-30天** | 建立"错题本"（只记录做错的题和原因）；制定最后30天每日复习计划；明确弱项、制定突破目标 |
| **T-14天** | 停止学习新内容；重点复习错题本；进行一次全真模考 + 复盘 |
| **T-7天** | 每日轻度复习（2小时以内）；重点练习口语（每天10分钟）；调整作息（固定起床和睡觉时间） |
| **T-1天** | 完全不学习；准备好考试物品：2B铅笔、橡皮、准考证、水、手表（非智能）；早点休息，保证睡眠 |
| **T-Day（考试当天）** | 早起 → 轻吃早餐（不要吃太饱）→ 检查物品 → 提前30分钟到达考场 → 深呼吸，对自己说"I can do it" → 考场内：先填姓名考号 → 按平时节奏做题 → 做完一定要检查 → 自信交卷 |

### 焦虑缓解技巧

**4-7-8 呼吸法：**
- 吸气4秒
- 屏住呼吸7秒
- 缓慢呼气8秒
- 重复3次

**正向心理暗示：**
- "我已经准备了很久，我可以做到的"
- "KET只是一个测试，我已经掌握了该会的知识"
- "遇到不会的题很正常，跳过做下一题"

**考前5分钟破冰话术（口语Part 1前）：**
- 深呼吸一次
- 微笑（即使紧张，微笑可以缓解紧张）
- 对自己说"I'm ready"
