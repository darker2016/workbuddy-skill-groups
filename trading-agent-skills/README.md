# 交易分析团队 Skills 文档体系

生成日期：2026-07-02

## 概述

本套 SKILL.md 文档完整还原了交易分析团队的 13 个角色技能，覆盖从数据收集到最终报告的 5 阶段工作流。每个 skill 包含完整的角色定位、输入规范、分析框架、产出格式和工作原则。

## 工作流结构

```
Phase 1（4人并行）→ Phase 2（顺序辩论）→ Phase 3（交易员）→ Phase 4（3人并行+裁决）→ Phase 5（整合报告）
```

## 数据源

所有成员均使用 **NeoData 金融数据服务**（neodata-financial-search skill），禁止使用 Yahoo Finance、Tushare、Bloomberg 等其他数据源。
