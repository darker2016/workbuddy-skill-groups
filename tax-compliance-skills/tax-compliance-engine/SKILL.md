# 财税合规知识引擎

本知识引擎为财税合规专家团所有成员（钱合规、李票务、张记账、王报表、孙申报、赵审计）提供共享的财税知识支持。成员在执行各自任务时，如需查阅基础数据、公式、标准或法规，可直接引用本引擎内容。

## 三、关键财务公式

### 盈利能力
- 毛利率 = (营业收入 - 营业成本) / 营业收入 × 100%
- 净利率 = 净利润 / 营业收入 × 100%
- 总资产报酬率(ROA) = 净利润 / 平均总资产 × 100%
- 净资产收益率(ROE) = 净利润 / 平均净资产 × 100%

### 偿债能力
- 流动比率 = 流动资产 / 流动负债
- 速动比率 = (流动资产 - 存货) / 流动负债
- 资产负债率 = 总负债 / 总资产 × 100%
- 利息保障倍数 = 息税前利润(EBIT) / 利息费用

### 营运能力
- 应收账款周转率(次) = 营业收入 / 平均应收账款
- 存货周转率(次) = 营业成本 / 平均存货
- 总资产周转率(次) = 营业收入 / 平均总资产

### 税务
- 增值税税负率 = 应纳增值税额 / 不含税销售收入 × 100%
- 所得税贡献率 = 应纳所得税额 / 营业收入 × 100%
- 综合税负率 = 各项税费合计 / 营业收入 × 100%

## 四、关键合规检查清单

### 发票合规
- [ ] 发票抬头是否与企业全称一致（含税号）
- [ ] 专票与普票是否按规定区分使用
- [ ] 发票内容与实际交易是否一致
- [ ] 进项发票是否在规定期限内认证（360天）
- [ ] 是否存在虚构交易/虚开发票
- [ ] 走逃失联发票是否及时处理
- [ ] 大额发票是否有对应合同/付款凭证

### 账务合规
- [ ] 记账凭证是否附有合规原始凭证
- [ ] 借贷是否平衡
- [ ] 科目使用是否符合会计准则
- [ ] 期末计提/摊销是否完整
- [ ] 往来款是否定期核对清收
- [ ] 存货盘点差异是否及时处理
- [ ] 固定资产折旧方法是否一致
- [ ] 坏账准备是否足额计提

### 税务合规
- [ ] 增值税纳税义务发生时间是否准确
- [ ] 进项税额转出是否完整
- [ ] 视同销售是否申报纳税
- [ ] 所得税纳税调整是否完整
- [ ] 关联交易定价是否公允
- [ ] 优惠资格是否持续满足条件
- [ ] 各税种申报是否按时完成
- [ ] 汇算清缴是否按期办理
