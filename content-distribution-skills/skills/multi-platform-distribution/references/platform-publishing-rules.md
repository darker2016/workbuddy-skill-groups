# 多平台发布规则完整参考

| TikTok | API 自动发布 | OAuth2 | 短视频 | 9:16优先、≤10分钟、Webhook回调确认 |
| Facebook | API 自动发布 | OAuth2 (Meta) | 视频/图文/Reels/Stories | 多种内容类型、Facebook Groups 支持 |
| Instagram | API 自动发布 | OAuth2 (Meta) | Reels/帖子/Stories/Carousel | 正方形/竖版优先、通过 Meta Graph API |
| Threads | API 自动发布 | OAuth2 (Meta) | 文字/图文 | 与 Instagram 关联账号，文字为主 |

## 二、标准发布流程 SOP（五步法）

### Step 1: 查询平台发布限制
发布前必须先查询目标平台的当前限制条件：
- 内容格式要求（尺寸/时长/文件大小/编码格式）
- 文案长度限制
- 标签数量和格式要求
- 当前是否有特殊限制（如节假日审核加严）

### Step 2: 检查账号可用性
1. 获取账号组列表
2. 按组获取账号列表
3. 筛选匹配目标平台的账号
4. 验证账号认证状态是否有效

账号信息格式：`[平台类型] 账号名称 (id: xxx)`

### Step 3: 账号决策树
```
发现目标平台账号？
├─ 否 → 引导用户绑定/创建该平台账号
└─ 是 → 检查认证状态
         ├─ 认证过期 → 引导用户重新授权
         └─ 认证有效 → 进入发布流程
```

### Step 4: 执行平台特定发布
- API 自动平台：携带 accountId + 内容元数据调用发布接口
- 手动平台：保存草稿后提示用户到对应 App 完成手动发布

### Step 5: 结果处理
| 发布结果 | 处理方式 |
|---------|---------|
| 成功 | 记录发布ID(flowId)，标记为已完成 |
| 账号未找到 | 引导创建/绑定账号 |
| 认证过期 | 引导重新授权 |
| 发布失败 | 记录错误信息，提供重试建议 |

## 三、内容元数据标准

### 必填字段
| 字段 | 要求 | 说明 |
|------|------|------|
| title | ≤100字符 | 包含核心关键词，吸引点击 |
| description | 平台适配 | 体现品牌调性 |
| medias | 至少1个 | 视频/图片的URL列表 |
| platform | 目标平台 | YOUTUBE/TIKTOK/BILIBILI 等 |

### 可选字段
| 字段 | 说明 |
|------|------|
| tags | #话题标签，5-15个 |
| topics | 平台分类标签（B站分区、YouTube类目等） |
| coverUrl | 自定义封面图URL |
| publishTime | 定时发布时间 |
| accountId | 指定发布账号 |

## 四、各平台内容规格速查

### 抖音
- **视频比例**: 9:16（竖版优先），也支持 16:9、1:1
- **视频时长**: ≤15分钟（建议15-60秒最佳表现）
- **视频格式**: MP4、MOV
- **封面**: 1:1 或 9:16，建议 720×1280 以上
- **标题**: ≤55字符（含空格）
- **话题**: #标签，3-5个
- **特殊**: 无完整自动化API，需草稿→手动发布

### 小红书
- **图片比例**: 1:1（正方形）、3:4（竖版）
- **图片数量**: 1-18张（推荐6-9张最佳）
- **视频比例**: 1:1、3:4、9:16
- **视频时长**: ≤15分钟（建议1-3分钟）
- **标题**: ≤20字符
- **正文**: ≤1000字符
- **话题**: 搜索关键词标签，10-15个
- **特殊**: 无自动化API，审核严格，禁止明显广告

### 快手
- **视频比例**: 9:16 竖版优先
- **视频时长**: ≤10分钟
- **视频格式**: MP4
- **标题**: ≤40字符
- **认证**: OAuth2 自动发布

### B站 (Bilibili)
- **视频比例**: 16:9（横版主流）
- **视频时长**: ≤4小时（长视频友好）
- **视频格式**: MP4、FLV
- **标题**: ≤80字符
- **描述**: ≤2000字符
- **标签**: 3-10个，需选择分区
- **发布方式**: API + Webhook回调

### TikTok
- **视频比例**: 9:16（强制竖版）
- **视频时长**: ≤10分钟（建议15-60秒）
- **描述**: ≤2200字符
- **发布方式**: OAuth2 API + Webhook回调
- **特殊**: 各地区音乐版权不同

### YouTube
- **视频比例**: 16:9（标准），Shorts为9:16（≤60秒）
- **视频时长**: 无上限（建议8-15分钟为主力内容）
- **标题**: ≤100字符
- **描述**: ≤5000字符（前2行最关键）
- **标签**: ≤500字符总长
- **缩略图**: 1280×720（16:9），≤2MB

### Instagram
- **图片比例**: 1:1、4:5（竖版推荐）、16:9
- **Reels比例**: 9:16
- **Reels时长**: ≤90秒
- **Carousel**: 最多10张图/视频
- **描述**: ≤2200字符
- **标签**: ≤30个（建议20-25个）

### X (Twitter)
- **文字**: ≤280字符
- **图片**: 16:9 或 1:1，最多4张
- **视频**: ≤2分20秒

### Pinterest
- **Pin图片比例**: 2:3（竖版优先，1000×1500px推荐）
- **视频Pin**: 2:3 或 9:16，≤15分钟
- **标题**: ≤100字符
- **描述**: ≤500字符（搜索SEO关键）
- **Board**: 必须归入一个Board

### LinkedIn
- **视频比例**: 16:9（横版专业内容）
- **视频时长**: ≤10分钟
- **文章**: 无字数上限
- **帖子**: ≤3000字符
- **图片**: 1200×627（推荐）
- **特殊**: 外链降权，优先发原生内容

### Facebook
- **视频比例**: 多比例支持，Reels为9:16
- **视频时长**: 建议≤3分钟
- **图片**: 最多10张

### Threads
- **文字**: ≤500字符
- **图片**: 最多10张
- **视频**: ≤5分钟
- **特殊**: 与Instagram账号关联

### 微信视频号
- **视频比例**: 9:16（竖版推荐）、16:9（横版支持）
- **视频时长**: ≤30分钟（建议1-3分钟表现最佳）
- **描述**: ≤1000字符
- **话题标签**: 3-5个（#标签格式）
- **定位**: 支持添加地理位置标签
- **发布方式**: 创作者中心手动、视频号助手（PC端）
- **直播**: 无时长限制，支持挂小程序商品链接

## 五、错误处理与重试策略

| 错误类型 | 原因 | 解决方案 |
|---------|------|---------|
| AUTH_EXPIRED | 令牌过期 | 引导用户重新授权 |
| RATE_LIMITED | 触发频率限制 | 等待后重试（指数退避） |
| CONTENT_REJECTED | 内容审核不通过 | 修改违规内容后重试 |
| FILE_TOO_LARGE | 文件超出限制 | 压缩后重试 |
| FORMAT_INVALID | 格式不支持 | 转码后重试 |
| NETWORK_ERROR | 网络问题 | 自动重试（最多3次） |

### 重试策略
- 首次重试：等待 30 秒
- 第二次重试：等待 60 秒
- 第三次重试：等待 120 秒
- 超过3次：标记失败，人工介入
