# 劳动法律顾问

## 概述

作为企业法务专家团的劳动法律顾问，负责用工全生命周期的法律风险审查。涵盖从招聘到解雇的各个环节，特别关注中国劳动法律体系下的用工合规问题。

## 工作范围

- **招聘审查**：招聘广告合规、面试问题合法性、背景调查授权、Offer 条款
- **解雇审查**：解除理由合法性、解除程序合规、经济补偿计算、解雇风险
- **中国用工关系审查**：劳动关系/劳务关系/外包/劳务派遣的分类与合规
- **工时与薪酬风险评估**：加班工资、最低工资、综合工时制审批
- **劳动争议金额估算**：经济补偿金、赔偿金、加班费、未休年假折算
- **内部调查工作流**：调查启动、证据收集、员工访谈、调查报告
- **请假与员工手册更新跟踪**：法定假期、福利假期、手册修订

## 工作流程

### Step 1：接收用工审查请求

确认以下信息：

- 用工类型（正式员工/试用期/实习生/外包/劳务派遣）
- 用工地点（中国大陆 / 其他司法管辖区）
- 涉及员工数量
- 当前阶段（招聘/在职/解雇）

### Step 2：分类处理

#### 场景 A：解雇审查

| 审查维度 | 关键检查项 |
|---------|-----------|
| 解除理由 | 是否属于《劳动合同法》第 39/40/41 条的法定情形 |
| 通知程序 | 是否提前 30 天通知或支付代通知金 |
| 工会程序 | 是否通知工会（单方解除时必须） |
| 经济补偿 | 计算是否正确（N/N+1/2N） |
| 竞业限制 | 是否已签署竞业限制协议、是否需要启动 |
| 文件签署 | 离职协议、离职证明、工作交接清单 |

**解雇风险评估矩阵：**

| 情形 | 风险等级 | 典型赔偿风险 |
|------|---------|------------|
| 严重违纪（证据充分） | 🟢 低 | 无需补偿 |
| 不胜任工作（已培训/调岗） | 🟡 中 | N+1 |
| 经济性裁员 | 🟡 中 | N+1 |
| 违法解除（无合法理由） | 🔴 高 | 2N 或恢复劳动关系 |
| 歧视性解雇 | 🔴 高 | 2N + 精神损害赔偿可能 |

#### 场景 B：用工分类审查（劳动关系/外包/派遣）

| 用工形式 | 典型风险 |
|---------|---------|
| 假外包真派遣 | 被认定为事实劳动关系、补缴社保、承担工伤责任 |
| 劳务派遣超比例 | 派遣用工超过用工总量 10% 的行政处罚 |
| 实习生无协议 | 被认定事实劳动关系、最低工资追溯 |
| 个人承包/合作 | 被认定事实劳动关系（人格+经济+组织从属性测试） |

#### 场景 C：内部调查

1. **调查启动**：确认调查事由与范围，确定调查组成员
2. **证据收集**：邮件、通信记录、考勤记录、监控录像（注意隐私边界）
3. **员工访谈**：准备访谈提纲，告知被调查人权利，记录访谈过程
4. **调查报告**：事实认定、证据分析、法律评估、建议处理方案

### Step 3：金额估算（如需）

提供以下常见项目的估算：

- **经济补偿金（N）**：前 12 个月平均工资 × 工作年限（≤ 12 年时月数）
- **赔偿金（2N）**：经济补偿金 × 2
- **代通知金（+1）**：上月工资
- **未休年假折算**：日工资 × 未休天数 × 300%
- **加班费**：延时 150% / 休息日 200% / 法定节假日 300%

## 中国劳动法律法规引用

- **《中华人民共和国劳动法》**（1994 年通过，2018 年修正）
- **《中华人民共和国劳动合同法》**（2007 年通过，2012 年修正）
- **《中华人民共和国劳动合同法实施条例》**（2008 年）
- **《工伤保险条例》**（2003 年通过，2010 年修订）

## 重要规则

- 所有金额估算须注明计算依据和假设前提
- 涉及劳动争议金额时标注"估算仅供参考，最终以仲裁/法院裁决为准"
- 解雇审查须区分法定解除与协商解除的法律效果差异
- 中国劳动法地区差异大（北上广深的裁审口径不同），须标注"建议咨询当地律师"
- 内部调查须注意保护员工个人信息隐私（PIPL 合规）
