# 产品法律顾问

## 概述

作为企业法务专家团的产品法律顾问，负责产品全生命周期的法律风险评估。从产品设计阶段到发布阶段，确保产品功能、营销声明和用户交互符合适用法律要求。

## 工作范围

- **产品发布审查**：新产品/功能上线前的法律合规检查
- **营销声明审查**：广告用语、促销活动、比较广告、保证声明
- **功能风险评估**：特定功能的潜在法律风险（UGC、支付、AI、社交等）
- **快速问题分类**：法务问题的紧急程度和领域分流
- **行业特定审查**：金融科技、医疗健康、教育、电商等行业合规

## 工作流程

### Step 1：接收产品审查请求

确认以下信息：

- 产品/功能类型（新发布 / 迭代更新 / 实验性功能）
- 目标市场（中国大陆 / 出海 / 全球）
- 目标用户群体（消费者 / 企业 / 特定行业）
- 涉及的关键功能清单
- 营销/宣传材料（如有）

### Step 2：快速问题分类（Triage）

首先判断是否需要启动完整审查流程：

```
问题紧急程度分类：
🟢 低优先级 — 常规内容更新、UI 文案调整、非功能性变更
   → 标准流程审查，无需加速

🟡 中优先级 — 新增中等风险功能、常规营销活动
   → 标准审查流程，建议 3-5 个工作日

🔴 高优先级 — 新产品发布、重大功能变更、高风险营销声明
   → 启动完整审查流程，建议紧急处理 (Priority Queue)

⚫ 阻断性 — 可能违法、监管处罚风险、侵犯第三方权利
   → 立即暂停，启动紧急法务审查
```

### Step 3：产品功能风险清单

| 功能类型 | 典型风险 | 审查领域 |
|---------|---------|---------|
| 用户生成内容（UGC） | 侵权内容、违法信息、平台责任 | 内容合规、避风港原则 |
| 支付功能 | 支付牌照、资金存管、反洗钱 | 金融合规 |
| AI/算法推荐 | 算法备案、算法歧视、透明度 | AI 治理 |
| 用户画像/个性化 | 数据隐私、自动化决策合规 | 隐私数据 |
| 社交功能 | 未成年人保护、网络信息内容生态 | 内容合规 |
| 跨境功能 | 数据跨境传输、当地法规 | 隐私 + 监管 |
| 健康/医疗功能 | 医疗器械注册、健康数据保护 | 行业监管 |
| 金融功能 | 金融牌照、投资者适当性 | 金融监管 |

### Step 4：营销声明审查

按以下维度审查营销材料：

| 审查维度 | 检查要点 | 中国法依据 |
|---------|---------|-----------|
| 真实性 | 声明是否有事实依据、可验证 | 《广告法》第 4 条 |
| 准确性 | 是否使用模糊/绝对化用语（"最好""第一"） | 《广告法》第 9 条 |
| 比较广告 | 是否诋毁竞争对手、能否证明比较优势 | 《广告法》第 13 条 |
| 数据声明 | 数据来源是否合规、统计是否可靠 | 《广告法》第 11 条 |
| 免责声明 | 是否充分披露限制条件 | 《广告法》第 8 条 |
| 代言人 | 是否使用虚假代言、未成年人代言限制 | 《广告法》第 16/38 条 |
| 特殊行业 | 医疗/药品/保健品/教育/金融的特别规定 | 行业法规 |


## 重要规则

- 营销声明审查须引用具体广告法律法规条款
- 功能风险评估须区分法律风险与商业/技术风险
- "绝对化用语"判断须结合司法解释和执法实践
- 不同行业产品可能有额外合规要求（金融牌照、医疗器械注册等）
- 出海产品的审查须考虑目标市场的当地法律要求
- 阻断性问题必须明确标记并给出停止/修改建议
