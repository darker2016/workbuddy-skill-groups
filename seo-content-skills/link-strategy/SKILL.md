# 链接策略与内容健康度工具

## 功能说明

提供系统化的链接策略设计框架，通过战略性内链布局建立主题权威性，改善用户导航体验，并提升页面搜索权重。涵盖内链架构、外链推荐、主题聚类连通和内容健康度评估。

## 核心分析模块

### 1. 内链机会分析

**链接放置策略**：

| 位置 | 优先级 | 说明 |
|------|--------|------|
| 正文上下文链接 | ⭐ 核心重点 | 在自然语境中引入相关页面 |
| 引言链接 | 少量使用 | 引出背景或前置知识 |
| 结论链接 | 下一步引导 | 引导读者继续深入 |
| 列表项链接 | 天然适配 | 在枚举中自然引入 |

**链接数量指南**：
- 最少：3 条
- 最优：4-5 条
- 最多：7 条（仅 3000+ 词文章）

**分配建议**：
- 支柱内容：1-2 条链接
- 相关博文：2-3 条链接
- 产品/工具页：1 条链接
- 资源页面：0-1 条链接

**避免的做法**：
- 无价值的强制链接
- 单段落超过 1-2 个链接
- 仅链接产品/销售页面
- 同一链接重复出现多次
- 模糊的锚文本（"点击这里"、"了解更多"）

### 2. 锚文本优化

**最佳实践**：
- 描述性：读者不点击也知道目标页面内容
- 自然流畅：不打断阅读节奏
- 关键词丰富：包含目标关键词但不堆砌
- 多样化：同类内容避免使用相同锚文本
- 适当长度：通常 2-5 个词

**锚文本类型与使用频率**：

| 类型 | 使用频率 | 示例 |
|------|----------|------|
| 部分匹配 | 最常用 | "提升网站转化率的方法" |
| 精确匹配 | 少量使用 | "SEO 优化指南" |
| 品牌型 | 适量 | "[品牌名]平台功能" |
| 自然型 | 适量 | "这份完整指南" |
| 裸URL | 极少 | 仅在特殊情况 |

### 3. 主题聚类链接架构

**聚类链接规则**：
- 支撑文章必须链回 Pillar 页面
- Pillar 页面应链向 2-3 篇聚类文章
- 支撑文章间可交叉链接（共同主题时）
- 创建相关内容的互联网络

**权重流向设计**：
```
Pillar Page (主权重节点)
├── 支撑文章 A ←→ 支撑文章 B
├── 支撑文章 C ←→ 支撑文章 D
└── 支撑文章 E
     ↑
     └── 外部链接入口
```

### 4. 外链策略

**目标：每篇 2-3 条外部链接**

**选择标准**：
- Domain Authority 高的权威来源
- 与内容主题直接相关
- 非竞品网站
- 链接到具体数据/研究/工具（而非首页）
- 来源时效性（优先近 2 年内容）

**适合外链的内容类型**：
- 行业研究报告和白皮书
- 权威统计数据源
- 开源工具或资源
- 学术研究或官方文档
- 行业标准和规范

### 5. 内容健康度评分（0-100）

| 维度 | 权重 | 检查项 |
|------|------|--------|
| 链接完整性 | 25% | 内链数量、外链数量、是否有断链 |
| 锚文本质量 | 20% | 描述性、多样性、关键词相关性 |
| 聚类连通性 | 20% | 是否链回 Pillar、与相关文章互联 |
| 链接分布 | 15% | 链接是否均匀分布在全文各段 |
| 用户价值 | 10% | 每条链接是否对读者有帮助 |
| 竞品对标 | 10% | 链接策略是否达到行业标准 |

### 6. 竞品差距分析

**分析维度**：
- 竞品文章的内链数量和策略
- 竞品引用的外部来源
- 竞品的聚类覆盖范围
- 我方可抢占的链接机会
- 竞品未覆盖的权威来源

## 输出格式

### 链接策略报告模板

```markdown
#### 文章概览
- **文章主题**：[主题]
- **当前内链数**：[X] 条
- **当前外链数**：[X] 条
- **内容健康度评分**：[X/100]

#### 推荐内链（3-5 条）
| # | 目标页面 | 锚文本 | 插入位置 | 理由 |
|---|----------|--------|----------|------|

#### 推荐外链（2-3 条）
| # | 来源 | 链接内容 | 插入位置 | 权威性说明 |
|---|------|----------|----------|------------|

#### 用户旅程链接地图
读者入口 → [当前文章]
    → 深入了解: [推荐内链A]
    → 相关话题: [推荐内链B]
    → 实践工具: [推荐内链C]
    → 下一步行动: [推荐内链D/CTA]

#### 实施清单
- [ ] 添加 [X] 条内链
- [ ] 添加 [X] 条外链
- [ ] 验证所有链接可访问
- [ ] 锚文本通过自然度检查
- [ ] 链接分布均匀
- [ ] 聚类连通性达标
```

## 质量标准

**每条链接必须**：
1. 为读者提供真实价值
2. 与上下文高度相关
3. 有清晰的存在目的
4. 使用描述性锚文本
5. 自然融入文本流
6. 支持 SEO 目标
7. 不过度链接

**评估问题**：
- 读者真的会点击这个链接吗？
- 这条链接是否帮助读者完成目标？
- 这是不是链接到这个主题的最佳页面？
- 锚文本是否足够清晰？
- 前文是否已经链接过同一页面？
- 是否过于推销？

## 指导原则

1. **用户优先**：链接服务读者旅程，不是 SEO 堆砌
2. **质量 > 数量**：3 条精准链接 > 7 条随意链接
3. **自然整合**：链接不应打断阅读体验
4. **战略放置**：每条链接都有明确的战略目的
5. **聚类意识**：始终从聚类全局视角思考链接
6. **转化导向**：在不牺牲用户体验的前提下引导转化路径
7. **分布均衡**：链接不扎堆在某一段，全文均匀分布

## 数据获取方式

- 使用 **Read** 读取待分析的文章稿件
- 使用 **WebSearch** 搜索竞品文章的链接策略和权威外链来源
- 使用 **WebFetch** 抓取推荐外链目标页面
- 使用 **Glob** 扫描项目中已有内容，发现内链机会
