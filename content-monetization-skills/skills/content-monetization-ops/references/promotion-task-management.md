# 推广任务全生命周期管理

## 概述

推广任务是内容商业化平台的核心业务单元，连接品牌方的推广需求与创作者的内容生产能力。本文档定义了推广任务从创建到结算的完整生命周期管理规范。

---

## 一、任务生命周期（7步流程）

### Step 1: 任务创建

**商家发布需求，定义推广任务**

必填信息：
| 字段 | 说明 | 示例 |
|------|------|------|
| 任务标题 | 简明任务名称 | "春季新品种草推广" |
| 产品信息 | 推广产品详情 | 品名/规格/卖点/价格 |
| 推广目标 | 核心KPI | 互动量>5000/转化>100单 |
| 预算总额 | 本次任务总预算 | 50000元 |
| 单人预算 | 每位创作者预算 | 5000元/人 |
| 内容要求 | Brief创作指南 | 视频≥60秒/需露出产品 |
| 素材包 | 提供的品牌素材 | 产品图/logo/文案参考 |
| 执行周期 | 任务时间范围 | 2024.03.01-2024.03.15 |
| 目标平台 | 指定发布平台 | 小红书+抖音 |
| 结算模式 | CPS/CPE/CPM | CPE，3元/互动 |
| 招募人数 | 需要的创作者数 | 10人 |
| 创作者要求 | 门槛条件 | 粉丝>5万/美妆品类/互动率>3% |

可选信息：
- 竞品排他（不可同期推竞品）
- 修改轮次上限（默认2轮）
- 独家要求（期限和范围）
- 内容审核标准补充

### Step 2: 任务匹配

**系统推荐 + 创作者主动接单，双向匹配**

匹配路径：
1. **系统推荐**（主动匹配）：
   - 基于匹配算法自动推荐符合条件的创作者
   - 推送至创作者任务中心（Top 20 匹配结果）
   - 推荐排序：匹配度>响应速度>近期活跃度

2. **创作者主动接单**（被动匹配）：
   - 任务广场展示所有开放任务
   - 创作者浏览筛选后主动报名
   - 商家从报名者中选择

3. **匹配确认流程**：
   - 创作者表达意向 → 商家审核创作者资质 → 双方确认 → 锁定合作
   - 确认后 24h 内双方不可单方面取消（冷静期后）

### Step 3: 内容创作

**创作者按 Brief 要求生产内容**

创作规范：
- 严格遵循 Brief 中的内容要求
- 修改轮次：默认最多 2 轮修改
- 首稿交付时限：确认合作后 3-5 个工作日
- 修改响应：商家反馈后 2 个工作日内提交修改版

质量标准：
- 产品露出时长/面积达标
- 品牌元素（logo/slogan）正确呈现
- 内容真实性（非虚假评测）
- 技术质量（画质/音质/剪辑）达标
- 无违规内容（不触碰平台红线）

### Step 4: 审核确认

**商家审核内容质量**

审核规则：
- 商家须在收到内容后 48h 内完成审核
- 超时未审核 → 视为自动通过
- 审核结果：通过 / 需修改（附具体修改意见）/ 驳回（附详细理由）
- 驳回后创作者可申诉（平台介入仲裁）

审核维度：
| 维度 | 权重 | 合格标准 |
|------|------|---------|
| Brief 契合度 | 30% | 核心要求全覆盖 |
| 产品呈现 | 25% | 露出时长/面积达标 |
| 内容质量 | 25% | 画质/剪辑/文案达标 |
| 合规性 | 20% | 无违规/无竞品/有广告标识 |

### Step 5: 发布执行

**创作者发布到指定平台**

发布要求：
- 在约定时间内发布到指定平台
- 发布后提交截图凭证（发布截图+数据截图）
- 不得在发布后 24h 内删除/隐藏内容
- 内容保留期：默认 30 天（合同另约除外）

发布凭证：
- 发布链接/URL
- 发布时间截图
- 初始数据截图（发布后 1h）
- 平台发布成功提示截图

### Step 6: 效果追踪

**按追踪期监测数据表现**

追踪周期：
| 类型 | 追踪期 | 关键节点 |
|------|--------|---------|
| 短效果 | 7天 | 48h数据/7天累计 |
| 中效果 | 14天 | 7天数据/14天累计 |
| 长效果 | 30天 | 14天数据/30天累计 |

追踪指标（按模式）：
- CPS：点击量、加购量、成交量、成交金额、退货量
- CPE：点赞数、评论数、分享数、收藏数、有效互动总量
- CPM：曝光量、独立触达人数、频次分布

数据采集方式：
- API 自动采集（已对接平台）
- 创作者手动提交后台截图（未对接平台）
- 第三方监测工具验证（大型campaign）

### Step 7: 结算清分

**达标后按约定模式结算**

结算触发条件：
- 追踪期结束
- KPI 达标（或部分达标）
- 无未解决争议
- 商家已确认数据

结算计算：
```
CPS 结算额 = 有效成交金额 × 佣金比例 - 退货扣除
CPE 结算额 = 有效互动总量 × 互动单价
CPM 结算额 = 有效曝光量 / 1000 × CPM单价
实际到手 = 结算额 × (1 - 平台服务费率) - 代扣税费
```

## 二、任务类型矩阵

| 类型 | 创作要求 | 结算模式 | 适用品牌 | 预算参考 | 效果预期 |
|------|---------|---------|---------|---------|---------|
| 产品种草 | 原创体验分享 | CPS/CPE | 新品推广 | 500-5000元/条 | 互动率>5% |
| 品牌曝光 | 植入品牌元素 | CPM | 品牌认知 | 1000-10000元/条 | 曝光>10万 |
| 活动转化 | 引导购买/注册 | CPS | 促销活动 | 按转化分成 | 转化率>2% |
| 口碑维护 | 正面评价/好评 | CPE | 口碑管理 | 200-2000元/条 | 正面率>90% |
| 矩阵分发 | 批量发布 | CPM | 铺量覆盖 | 50-500元/条 | 覆盖>100万 |
| 深度评测 | 专业对比评测 | 固定费 | 高价品 | 5000-50000元/条 | 可信度>4.5 |
| 直播带货 | 直播间介绍 | CPS | 即时转化 | 坑位费+分成 | GMV>10万 |
| 话题挑战 | 参与品牌话题 | CPE | 话题引爆 | 300-3000元/条 | 话题播放>500万 |

## 三、任务状态机

```
Draft(草稿)
  → Published(已发布): 商家提交发布
  
Published(已发布)
  → Matched(已匹配): 创作者确认接单
  → Cancelled(已取消): 商家主动取消
  → Expired(已过期): 超时无人接单

Matched(已匹配)
  → InProgress(进行中): 创作者开始创作
  → Cancelled(已取消): 任一方取消(冷静期内)

InProgress(进行中)
  → UnderReview(待审核): 创作者提交内容
  → Timeout(超时): 创作者超时未交付

UnderReview(待审核)
  → Approved(已通过): 商家审核通过
  → Revision(需修改): 商家要求修改
  → Rejected(已驳回): 商家驳回内容

Revision(需修改)
  → UnderReview(待审核): 创作者提交修改版
  → Timeout(超时): 修改超时

Approved(已通过)
  → Live(已发布): 创作者发布上线

Live(已发布)
  → Tracking(追踪中): 进入效果追踪期

Tracking(追踪中)
  → Settled(已结算): 追踪期满+达标
  → Dispute(争议中): 数据争议

Settled(已结算)
  → Completed(已完成): 款项到账

Dispute(争议中)
  → Settled(已结算): 仲裁完成
  → Cancelled(已取消): 仲裁判定取消
```

## 四、异常处理机制

### 4.1 创作者端异常

| 异常 | 触发条件 | 处理方式 | 影响 |
|------|---------|---------|------|
| 超时未交付 | 超过约定交付日期+48h | 自动取消任务 | 扣信用分10分+限接单7天 |
| 内容抄袭 | 被检测到高相似度 | 驳回+警告 | 严重者永久封禁 |
| 数据造假 | 互动数据异常 | 数据清洗+扣佣 | 扣信用分20分+冻结30天 |
| 擅自删帖 | 保留期内删除内容 | 扣除该笔结算 | 扣信用分15分 |
| 失联/拒绝沟通 | 48h未回复 | 警告→72h仍未回复自动取消 | 扣信用分5分 |

### 4.2 商家端异常

| 异常 | 触发条件 | 处理方式 | 影响 |
|------|---------|---------|------|
| 超时未审核 | 提交后48h未审核 | 自动通过 | 降低商家信用分 |
| 无理驳回 | 多次驳回无合理理由 | 平台介入仲裁 | 降信用分+限发任务 |
| 拖欠付款 | 结算后7天未付款 | 催付→冻结发任务权限 | 严重者封禁 |
| 频繁取消 | 30天内取消>3次 | 限发任务+需缴保证金 | 降低推荐权重 |

### 4.3 效果未达标处理

| 达标率 | 处理方式 | 结算比例 |
|--------|---------|---------|
| ≥100% | 正常结算+奖励 | 100%+奖金 |
| 80-99% | 正常结算 | 100% |
| 60-79% | 部分结算 | 按实际比例 |
| 40-59% | 协商处理 | 50%或重新执行 |
| <40% | 仲裁 | 视具体情况 |

**兜底机制**：
- 首次合作不达标：给予一次重新执行机会
- 非创作者可控因素（如平台限流）：不扣信用分
- 双方协商一致：可灵活调整结算方案
