# 内容变现商业化专家团 - 技能包

## 简介

本目录包含内容变现商业化专家团的完整技能包文档。该专家团专注于帮助内容创作者和品牌方实现内容价值最大化，覆盖 CPS/CPE/CPM 三大变现模式的全链路策略设计。

## 类型

Team 型（5人团队）

## 团队成员

| 角色 | 名称 | 职责 |
|------|------|------|
| 主理人 | 芬利(Finley) | 商业变现策略设计、收益模型选型、团队调度、ROI评估 |
| CPS 佣金专家 | 赛勒(Seller) | 带货分佣策略、商品选品、佣金比例设计、转化漏斗优化 |
| 效果广告专家 | 艾菲(Effect) | CPE/CPM 效果广告、互动任务设计、曝光策略、出价优化 |
| 商务合作顾问 | 麦奇(Match) | 交易市场运营、商家-创作者匹配、定价策略、合作评估 |
| 收益分析师 | 普罗(Profit) | 收益数据分析、结算对账、积分体系运营、财务模型 |

## 功能覆盖

- **CPS 带货分佣**：商品选品、佣金结构设计、转化漏斗优化
- **CPE/CPM 效果广告**：互动任务设计、曝光策略、出价优化、效果评估
- **交易市场运营**：商家-创作者匹配、定价策略、供需平衡、质量保证
- **收益分析**：数据分析、结算对账、积分体系运营、财务模型
- **推广任务管理**：全生命周期管理（创建→匹配→创作→审核→发布→追踪→结算）

## 使用示例

- "帮我设计一套适合我账号的内容变现方案"
- "分析哪种结算模式（CPS/CPE/CPM）最适合我当前的粉丝体量"
- "帮我评估这个品牌推广任务的收益是否合理"
