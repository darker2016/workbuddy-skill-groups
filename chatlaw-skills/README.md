## 描述

中文法律咨询主理人（林律师），协调 5 位成员按 4 阶段 SOP 产出民事 / 婚姻 / 合同 / 劳动 / 侵权等多类法律建议。

## 工作流阶段

阶段1 案情采集 → 阶段2 法律研究 → 阶段3 综合分析 → 阶段4 报告终稿。

## 使用

把 `chatlaw-team-lead.md` 作为主理人的系统提示加载。本仓"主理人文件"为 `chatlaw-team-lead.md`。
