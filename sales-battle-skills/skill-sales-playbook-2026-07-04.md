## 1. 客户调研 (Account Research)

**触发词**：调研公司、查客户、了解下这个客户

### 调研内容矩阵

| 维度 | 内容 | 来源 |
|------|------|------|
| 公司概况 | 做什么、规模、行业、融资 | 官网、企查查 |
| 近期动态 | 融资、管理层变动、产品发布 | 新闻、PR 稿 |
| 招聘信号 | 在招岗位 = 增长方向信号 | 招聘网站 |
| 关键人物 | 决策者、影响者、门禁人 | LinkedIn |
| 痛点假设 | 基于研究推断的业务挑战 | 综合推断 |

### 输出模板

```markdown
# 调研: [公司名称]
**日期:** [Date] | **来源:** 网络搜索

## 速览
[2-3句: 他们是谁，为什么可能需要你，最佳切入角度]

## 公司画像
| 字段 | 值 |
|------|---|
| 行业 | [X] | 规模 | [X] | 融资 | [X] | 总部 | [X] |

## 近期动态
- [新闻 + 与你的关联]

## 关键人物
| 姓名 | 职位 | 切入点 |

## 痛点假设 → 建议外联策略
```

## 7. 外联起草 (Draft Outreach)

**触发词**：写邮件、冷邮件、跟进邮件、LinkedIn 消息

### 五原则

1. **个性化** — 通用模板 = 垃圾邮件
2. **简短** — 首封不超过 5 句
3. **价值优先** — 先说对他们的价值
4. **明确 CTA** — 一个清晰行动号召
5. **不卑不亢** — 你在提供价值

### 邮件类型

**冷邮件**: [个性化主题] → [引用近期动态] → [点出痛点] → [价值主张] → [CTA]

**跟进邮件**: [新价值点，不是"想确认你看到了吗"] → [新切入点] → [CTA]

**LinkedIn 消息**: 3 句以内，随意语气

### 触达序列

| # | 时间 | 渠道 | 目的 |
|---|------|------|------|
| 1 | Day 0 | 邮件 | 初次 + 价值主张 |
| 2 | Day 3 | LinkedIn | 建立连接 |
| 3 | Day 7 | 邮件 | 新价值/案例 |
| 4 | Day 14 | 邮件 | 最后触达 |

## 8. 销售预测 (Forecast)

**触发词**：销售预测、本月能做多少、预测报告

### 预测分类

| 类别 | 定义 | 置信度 |
|------|------|--------|
| 已关闭 | 合同已签 | 100% |
| 承诺 | 口头同意 | 90% |
| 最佳情况 | 进展顺利 | 60% |
| Pipeline | 活跃，需推进 | 30% |
| 风险 | 有不确定性 | 10% |

### 输出模板

```markdown
## 销售预测 — [时间范围]
| 类别 | 金额 | 商机数 |
|------|------|--------|
| **加权预测** | **¥[X]** | |
| **目标** | **¥[X]** | |
| **差距** | **¥[X]** | |

### 风险分析 → 提升建议
```

## 9. Pipeline 审查 (Pipeline Review)

**触发词**：Pipeline 审查、商机清理、pipeline review

### 审查框架

| 因素 | 权重 |
|------|------|
| 预计关闭日期 | 30% |
| 金额 | 25% |
| 阶段 | 20% |
| 活跃度 | 15% |
| 风险 | 10% |

### 输出模板

```markdown
## Pipeline 审查 — [日期]

### 快照: 总商机 / 总金额 / 加权金额 / 平均周期 / 赢率

### 优先行动商机
| 商机 | 金额 | 阶段 | 下一步 | 风险 |

### 风险商机
| 商机 | 金额 | 风险信号 | 建议 |

### 数据清洁度 (缺日期/缺金额/缺下一步)

### 建议清除 (60天+无活动 → Closed-Lost)
```

**三原则**: 每周审查 / 果断清理死单 / 多线程联系人
